Available APIs

These APIs are running in demo mode with simulated responses. Payments are verified on Tempo Moderato testnet. Deploy your own with real upstreams using npx create-paygate.

Web Search

$0.002/request
POST /v1/search

Search the web. Returns titles, URLs, and descriptions.

{"query": "AI news"}

{"results": [{"title": "...", "url": "..."}]}

Web Scrape

$0.001/request
POST /v1/scrape

Extract clean text from any webpage as markdown.

{"url": "https://..."}

{"markdown": "# Article Title\n..."}

Image Generation

$0.01/request
POST /v1/image

Generate images from text prompts. Returns a URL.

{"prompt": "a cat astronaut"}

{"image_url": "https://..."}

Text Summarize

$0.003/request
POST /v1/summarize

Summarize long text into key points.

{"text": "Long article..."}

{"summary": "Key points..."}

Add your API to the marketplace

Three commands. Sixty seconds. Real payments.

$ npx create-paygate my-api
$ cd my-api
$ npm start
  • paygate.toml configures pricing and your wallet address
  • Dockerfile included for fly.io, Railway, or any container host
  • PayGate handles payments, verification, and proxying

For agent developers

One SDK. Automatic payment negotiation. Your agent handles 402s without human intervention.

import { PayGateClient } from '@paygate/sdk';

const client = new PayGateClient({ tempoClient });

// Automatic: discovers price, pays, retries
const result = await client.fetch(
  'https://paygate-demo-production.up.railway.app/v1/search',
  { method: 'POST', body: JSON.stringify({ query: 'AI news' }) }
);
