For agent builders

Your agent pays for itself.

It sees a 402, reads the price, pays on-chain, retries. You write zero payment code.

import { PayGateClient } from '@paygate/sdk';

const client = new PayGateClient({ tempoClient });

// That's it. Agent handles 402s automatically.
const result = await client.fetch(
  'https://api.example.com/v1/search',
  { method: 'POST', body: JSON.stringify({ query: 'AI news' }) }
);
npm install @paygate/sdk
