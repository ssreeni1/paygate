Quickstart

You hit an API. It returned 402 Payment Required. Here's what to do next.

TL;DR — Install the SDK and your agent handles payments automatically: 
npm install @paygate/sdk

What's happening

The API you called is gated by PayGate. It requires a small stablecoin payment per request — no API key, no account, no monthly bill. Just pay and get your response.

The 402 response tells you everything: the price, where to send it, and a quote_id that locks the price for 5 minutes.

HTTP/1.1 402 Payment Required

{
  "error": "payment_required",
  "pricing": {
    "amount": "0.002000",
    "amount_base_units": 2000,
    "token": "0x...USDC",
    "recipient": "0x...Provider",
    "quote_id": "qt_a1b2c3d4",
    "quote_expires_at": "2026-03-19T12:00:00Z"
  }
}

Recommended

Use the SDK

The SDK handles the full 402 flow automatically: detect payment required, pay on-chain, retry with proof.

1. Install

npm install @paygate/sdk

2. Create a client

import { PayGateClient } from '@paygate/sdk';
import { createClient, http, publicActions, walletActions } from 'viem';
import { Account, tempoActions } from 'viem/tempo';
import { tempoModerato } from 'viem/chains';

const account = Account.fromSecp256k1(process.env.TEMPO_PRIVATE_KEY);

const tempoClient = createClient({
  account,
  chain: tempoModerato,
  transport: http(),
})
  .extend(publicActions)
  .extend(walletActions)
  .extend(tempoActions());

const client = new PayGateClient({ tempoClient });

3. Make requests

Use it like fetch(). Payment happens transparently.

const response = await client.fetch(
  'https://api.example.com/v1/search',
  {
    method: 'POST',
    body: JSON.stringify({ query: 'AI news' }),
  }
);

const data = await response.json();

4. Use as an agent tool

const searchTool = {
  name: 'web_search',
  description: 'Search the web. Payment handled automatically.',
  execute: async ({ query }) => {
    const res = await client.fetch(
      'https://api.example.com/v1/search',
      { method: 'POST', body: JSON.stringify({ query }) }
    );
    return res.json();
  },
};

Pay manually with curl

For testing, debugging, or languages without an SDK.

1. Get the 402

curl -s https://api.example.com/v1/search \
  -X POST -H "Content-Type: application/json" \
  -d '{"query": "AI news"}'

Note the quote_id, amount, token, and recipient.

2. Compute the memo

The payment needs a memo that binds it to your request:

requestHash = keccak256(METHOD + " " + PATH + "\n" + BODY)
memo        = keccak256("paygate" + quoteId + requestHash)

3. Send TIP-20 payment on Tempo

Call transferWithMemo(to, amount, memo) on the TIP-20 token contract:

toThe recipient from the 402
amountAt least amount_base_units
memoThe bytes32 from step 2

4. Retry with proof

curl -s https://api.example.com/v1/search \
  -X POST -H "Content-Type: application/json" \
  -H "X-Payment-Tx: 0xYOUR_TX_HASH" \
  -H "X-Payment-Payer: 0xYOUR_WALLET" \
  -H "X-Payment-Quote-Id: qt_a1b2c3d4" \
  -d '{"query": "AI news"}'

PayGate verifies the payment on-chain (under 100ms) and proxies your request.

Response headers

After payment, your response includes:

X-Payment-Receipt Transaction hash. Look up at /paygate/receipts/{tx}.
X-Payment-Cost Amount charged (e.g., 0.002000 USDC).

All X-Payment-* headers are stripped before reaching the upstream API.

Run your own PayGate

Wrap your API in three commands:

npx create-paygate my-api
cd my-api
npm start

Edit paygate.toml to set your price and wallet address. Deploy anywhere that runs Docker.
